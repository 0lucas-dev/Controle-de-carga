# Projeto 3 Semestre da Faculdade
## Sobre o Projeto

Um sistema que gerencia e controla o fuxo de carga e descarga de vagas de estacionamente para melhorar o fluxo em certos horarios e não prejudicar o transito em areas muito movimentadas.

---

### Empresa de Apoio

[**DSIN**](https://www.dsin.com.br): Empresa que visa Facilidade, tecnologia e eficiência na gestão do trânsito.
